# ModpackManagementTools

A collection of tools to make the creation and distribution of Minecraft modpacks easier.
These tools are designed to allow the distribution of modpacks without requiring an upload to Curseforge, Modrinth, or
similar platforms.

## Tools

- **Modpack Exporter** – Package and distribute a modpack in various formats.
  See [Modpack Exporter](modpackExporter/README.md) for more information.
